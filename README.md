# JoAi Skills

Agent skills for the [Warp Protocol](https://joai.ai/warps) and JoAi platform.

## Installation

Install skills in one command using [skills.sh](https://skills.sh):

```bash
npx skills add JoAiHQ/skills
```

This adds the skills to your project for use with AI coding assistants like Cursor, Claude, and other MCP-enabled tools.
